# SVMG Matchday Post Generator

## 🚀 Deployment in 5 Minuten (kostenlos)

### Schritt 1: GitHub Account
1. Geh auf **github.com** → „Sign up" → kostenlosen Account erstellen

### Schritt 2: Neues Repository
1. Auf GitHub oben rechts auf **„+"** → „New repository"
2. Name: `svmg-matchday`
3. „Public" auswählen
4. „Create repository" klicken

### Schritt 3: Dateien hochladen
1. Klicke auf „uploading an existing file"
2. Lade **alle Dateien aus diesem Ordner** hoch (package.json, public/, src/)
3. „Commit changes" klicken

### Schritt 4: Vercel Account
1. Geh auf **vercel.com** → „Sign up" → „Continue with GitHub"

### Schritt 5: Deployen
1. Auf Vercel: „New Project" → dein `svmg-matchday` Repository auswählen
2. „Deploy" klicken
3. Nach ~2 Minuten bekommst du eine URL wie `svmg-matchday.vercel.app`

### Fertig! 🎉
- Die App läuft dauerhaft unter deiner eigenen URL
- localStorage funktioniert normal → alles wird gespeichert
- Kostenlos für immer
